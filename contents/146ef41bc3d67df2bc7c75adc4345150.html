CA-015 Controlador pagos

UMLClass «control»
Dentware :: Modelo de análisis :: Modelo de clases de análisis :: «control»CA-015 Controlador pagos

Description

Controlador que se encarga de recibir las peticiones de modificar datos de los pagos con todas las interfaces que corresponden al manejo de los pagos. Redirige las peticiones a CA-012 para obtener una respuesta.
