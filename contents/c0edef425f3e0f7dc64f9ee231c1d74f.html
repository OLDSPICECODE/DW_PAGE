CA-013 Controlador paciente

UMLClass «control»
Dentware :: Modelo de análisis :: Modelo de clases de análisis :: «control»CA-013 Controlador paciente

Description

Controlador que se encarga de recibir las peticiones de modificar datos de un paciente con todas las interfaces que corresponden al manejo de los pacientes. Redirige las peticiones a CA-012 para obtener una respuesta.
