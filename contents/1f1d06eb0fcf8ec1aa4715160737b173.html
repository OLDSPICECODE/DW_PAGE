Dibujar()

UMLMessage
Dentware :: Modelo de diseño :: Modelo de interacción :: SQ-005 Gestionar Odontograma :: Dibujar() (Usuario→Odontograma)

Description

La función Dibujar() gestiona todas las acciones relacionadas con la edición gráfica de la imagen del odontograma. Esta función permite seleccionar una herramienta de dibujo y realizar operaciones como trazado de líneas rectas, círculos, dibujo libre, inserción de texto, y gestión de acciones mediante una funcionalidad de deshacer. Cada acción realizada se almacena en una estructura de historial, lo que permite revertir múltiples acciones en orden inverso a su ejecución. De esta manera, el usuario puede retroceder en los cambios realizados, paso a paso, hasta llegar al estado deseado.
