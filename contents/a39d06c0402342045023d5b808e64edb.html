MR-008 odontograma

ERDEntity
Dentware :: Modelo de diseño :: Modelo de datos lógico :: MR-008 odontograma

Description

Registra gráficamente el estado dental del paciente. Para cada tratamiento deben existir tres odontogramas: el estado del paciente sin tratar, el tratamiento proyectado y un tercer odontograma que registre el avance del tratamiento.
