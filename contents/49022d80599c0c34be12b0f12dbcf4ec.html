MR-011 Paciente_pago

ERDEntity
Dentware :: Modelo de diseño :: Modelo relacional :: MR-011 Paciente_pago

Description

Registra los pagos realizados por el paciente en relación con un presupuesto aprobado. No representa pagos electrónicos ni boletas fiscales, sino un control interno del historial de pagos registrados manualmente por el personal administrativo.
