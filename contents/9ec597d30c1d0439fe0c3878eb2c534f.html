ActualizarDatos

UMLOperation
Dentware :: Modelo de diseño :: Modelo de clases de diseño :: «Control»CD-008 Tratamiento :: ActualizarDatos

Description

Función que actualiza un tratamiento existente modificando los campos como el estado actual del tratamiento (Activo, Finalizado), la fecha, descripción del tratamiento realizado, el profesional que lo realiza, y observaciones del tratamiento hechas por el profesional, pasando los datos ingresados a Entity Tratamiento.
