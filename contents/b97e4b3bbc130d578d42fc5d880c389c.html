CA-016 Controlador odontograma

UMLClass «control»
Dentware :: Modelo de análisis :: Modelo de clases de análisis :: «control»CA-016 Controlador odontograma

Description

Controlador que se encarga de recibir las peticiones de modificar datos de un tratamiento con todas las interfaces que corresponden al manejo de los tratamientos. Redirige las peticiones a CA-012 para obtener una respuesta.
