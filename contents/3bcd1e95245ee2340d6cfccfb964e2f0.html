CA-012 Controlador sistema

UMLClass «control»
Dentware :: Modelo de análisis :: Modelo de clases de análisis :: «control»CA-012 Controlador sistema

Description

Controlador principal del sistema, maneja el inicio de sesión, la búsqueda y la creación de pacientes. Es el encargado de coordinar los controladores que manejan el frontend con los controladores que manejan la base de datos.
