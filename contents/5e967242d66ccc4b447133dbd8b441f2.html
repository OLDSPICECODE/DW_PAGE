MR-006 tratamiento

ERDEntity
Dentware :: Modelo de diseño :: Modelo de datos lógico :: MR-006 tratamiento

Description

Representa el registro de la intervención odontológica para un paciente. Cada tratamiento está vinculado a un presupuesto aprobado y se compone de múltiples sesiones clínicas. Incluye información sobre el diagnóstico, estado del tratamiento y observaciones de los odontólogos tratantes.
