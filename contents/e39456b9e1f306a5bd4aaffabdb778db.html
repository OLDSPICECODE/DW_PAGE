MR-001 Paciente

ERDEntity
Dentware :: Modelo de diseño :: Modelo relacional :: MR-001 Paciente

Description

Contiene la información básica de los pacientes atendidos en la clínica, incluyendo datos personales y de contacto. Esta tabla es fundamental, ya que sirve como punto de referencia para asociar los tratamientos, presupuestos, exámenes, pagos y demás registros clínicos vinculados al historial del paciente.
